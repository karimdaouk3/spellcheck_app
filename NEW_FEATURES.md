# New Features in Version 1.0

This document outlines the new features added in Version 1.0 of FSR Coach, building upon the previous version's core functionality of case submission, review, and rewrite questions.

## New Features

- **Single Sign-On (SSO) Authentication**: Users authenticate via SSO with automatic user profile creation and management.
- **Case Tracking System**: All cases are tracked in a centralized database, allowing users to access their cases across sessions with automatic history tracking.
- **CRM Integration**: Integration with CRM systems to search, discover, and validate cases with automatic case title retrieval and status synchronization.
- **Persistent Case Data**: Case data automatically saves and persists across sessions, including problem statements and FSR notes.
- **Case Feedback System**: Mandatory feedback collection for closed cases to capture symptom, fault, and fix information.
